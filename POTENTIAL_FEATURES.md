# Microscape Calculator — Potential Features

## Yield / Batch Crafting
- Some crafting actions produce multiple items per action (e.g. 1 action → 10 items)
- Some items can be crafted in batches (e.g. submit 10 at once)
- Would add an optional **"Yield per action"** field on each item
- Only affects the **time estimate** — XP and craft count stay per-action as normal
- Bank trips would need to account for yield (bag fills faster if item is non-stackable)
- **On hold** until Chill Spot banking is designed, since stackable items skip the bag entirely and Chill Spot changes the banking model significantly
